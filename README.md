# YouTube Shopping — Executive Brief

Internal research dashboard on YouTube Shopping prepared for Trafilea board review.

🔗 **Live dashboard:** https://[your-username].github.io/[repo-name]/

## What's inside

A self-contained HTML dashboard (no build step, no dependencies beyond Chart.js loaded from CDN) covering:

- **Executive brief** — 5 KPIs of the YT Shopping ecosystem (35B hrs watched, 5× GMV YoY, 500K+ creators, 89% creator trust, USD 100B paid to creators)
- **What it is** — definition + 5 shopping surfaces (Channel Store, Product Shelf, Shopping Button, Product Stickers, Pinned Live)
- **How it works** — 3 operating models (Affiliate Program, Direct Store, BrandConnect)
- **The ecosystem in numbers** — 4 interactive Chart.js charts (growth, commissions, vs TikTok/IG, lifts)
- **Note for the board** — disclosure on geographic/device limitations for visualizing native tags from Latam
- **Creator cases** — 3 official testimonial videos (MsGoldgirl, MissEJLouie, VCG Construction) + 2 additional cases (ChicOnTheCheap, Steve Natto)
- **Brand cases** — 7 documented cases: BK Beauty, PillowtalkDerm (Dr. Idriss), HexClad, SACHEU Beauty, Glossier, Sephora, Nike
- **Trafilea perspective** — 4 actionable plays + 1 risks card
- **Sources** — 27 verifiable sources (official YouTube/Google + press)

## How to view

Open `index.html` in any modern browser, or visit the GitHub Pages URL above.

## How to share

The dashboard is self-contained and works offline (except for Chart.js CDN). You can:
- Share the GitHub Pages URL
- Download `index.html` and open it locally
- Print to PDF for static distribution

## Limitations

The native YouTube Shopping features (product tags, Shorts stickers, Shopping Button, in-app checkout) are **not visible from Latam or desktop browsers**. They only appear in the YouTube mobile app within eligible markets (US, India, Korea, Brazil, Indonesia, etc.). When clicking on the brand video links, you will see the videos but **not the product tags**. This is documented in section 05 of the dashboard.

The 3 creator testimonial videos in section 06 are recordings where the creators themselves explain the system, so they are 100% verifiable regardless of region.

## Tech notes

- Works offline except for the Chart.js CDN load on first paint
